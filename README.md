## 介绍

cain-video 一款基于vue3制作的video视频播放插件，ui美观，功能丰富，高度集成，持续更新

## 现有功能
------
1. 实现进度条功能
2. 实现画中画功能
3. 实现网页全屏
4. 实现浏览器全屏
5. 实现播放暂停
